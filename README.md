## 깍두기 볶음밥 레시피

### 재료
- 밥 한공기
- 깍두기
- 깍두기 국물
- 대파 약간
- 고기 약간
- 고추장 1스푼
- 고추가루 2스푼
- 진간장 3스푼

### 레시피 순서
1. 팬에 기름을 두르고 고기와 대파를 볶는다
2. 깍두기를 넣고 고추장, 고추가루 간장을 넣는다
3. 재료가 살짝 잠길 정도로 육수를 붓고 국물을 졸인다 `육수=깍두기 국물 1: 물 1 비율`
4. 국물이 쫄면 밥을 넣고 골고루 비벼 먹는다
5. **맛있게 먹는다~** 🍚 🥘
